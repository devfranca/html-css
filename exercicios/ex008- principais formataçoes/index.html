Principais Formatações

Negrito ou destaque

Nesta frase temos um termo em negrito usando a tag B (não semantica)

Nesta frase temos termo em destaque usando a tag STRONG (semantica)

Italico ou Enfase

Nesta frase temos um termo em italico usando a tag I (não semantica)

Nesta frase temos um termo em enfase usando a tag EM (semantica)

Texto marcado

Podemos criar tambem um texto marcado tambem usando a tag MARK

podemos tambem trocar a cor desse marca texto usando a tag STYLE

Texto grande e pequeno

estamos criando um texo grande e um texto pequeno usando a tag BIG apesar de ela ter sido desconsiderada, e SMALL.

Texto deletado

Podemos marcar um texto como excluido para ele ser lido, porem nao considerado, usando a tag DEL

Texto inserido

Podemos marcar um texto como inserido para dar enfase e indicar que ele foi adicionado depois, usando a tag INS

Existe tambem o sublinhado com a tag U (nao semantica)

Texto sobrescrito

Para inserir coisas do tipo x20+3, usando a tag SUP(em cima)

Texto subiscrito

Para inserir coisas do tipo H2o,usando a tag SUB(em baixo)
